# ✨ About the Project

Colorfly Studio is a web application designed to generate beautiful color palettes for designers, developers and creative people.

The project was created to practice JavaScript while focusing on user experience, clean design and intuitive interactions.

The main objective was to build a useful tool rather than simply completing a programming exercise.

# ✨ Features

- ✅ Generate random color palettes
- ✅ Lock individual colors
- ✅ Regenerate only unlocked colors
- ✅ Copy HEX and HSL values with one click
- ✅ Choose between 6 or 9 colors
- ✅ Responsive design
- ✅ Modern and intuitive interface
